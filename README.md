# Papyrs - CDN

A CDN for Papyrs.

## Content

- libraries that are loaded as ES modules with script tags
- assets: fonts, icons and social image

## Hosting

CDN is currently hosted on [Google Firebase](https://firebase.google.com).

Ultimately it will be hosted on the IC.
